PROJECT NAME:
Instagram Performance Analysis – Sugar Cosmetics

OVERVIEW:
This project analyses the Instagram performance of Sugar Cosmetics using Tableau.
The objective is to evaluate engagement, reach, content effectiveness, and posting
patterns, and to provide data-driven recommendations to improve organic social media
strategy.

TOOLS USED:
- Tableau (Data visualization & dashboarding)
- CSV file (Data source)

DATASET DETAILS:
- Platform: Instagram
- Time Period: April 2024 – April 2025 contains Post level data

KEY METRICS ANALYZED:
- Total Posts
- Total Likes
- Total Comments
- Total Reach
- Engagement Rate (calculated)
- Post Type Performance
- Hashtag Performance
- Day of Week Engagement

TASKS COVERED:
1. Overview KPIs – Instagram Summary
2. Monthly Posting Trend
3. Top 10 Posts by Likes
4. Top 10 Posts by Engagement Rate
5. Post Type Performance Comparison
6. Day of Week Analysis
7. Engagement Funnel
8. Hashtag Analysis
9. Interactive Dashboard
10. Strategic Recommendations

DASHBOARD FEATURES:
- Interactive filters (Post Date, Post Type, Campaign Type)
- KPI cards
- Business-friendly layout for marketing stakeholders

KEY INSIGHTS:
- Reels and Videos generate the highest engagement
- Fridays and Saturdays show peak engagement
- Ethical and brand-value hashtags(Group analysis) drive higher reach
- Engagement rate is a better for performance than likes.
